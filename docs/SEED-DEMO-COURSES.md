# Демо-курсы (сиды)

Скрипт: `scripts/seed-demo-courses.ts`

```bash
npm run seed:demo-courses
```

Создаёт **3 курса** по робототехнике (LEGO SPIKE, Arduino, FLL), в каждом **3 модуля** с цепочкой разблокировки (`unlock_after_module_id`), **блоки контента** (текст, плейсхолдеры под видео/картинку/файл, внешние ссылки) и **тесты** с вопросами.

Медиафайлы **не вшиваются**: в блоках `video` / `file` поле `file_url` пустое — загрузите файлы через админ-панель и пропишите URL (как после `POST /admin/upload/...`). Для иллюстраций в сиде используются **текстовые** блоки-подсказки (чтобы сид работал и без enum `image` в БД; при необходимости добавьте блок «Изображение» вручную). Обложки курсов (`thumbnail_url`) — `NULL`, загрузите при создании/редактировании курса.

Повторный запуск: если в БД уже есть курс с названием первого демо-курса, скрипт **ничего не делает**.

Пересоздать с нуля (удалит курсы с демо-названиями и связанные данные):

```bash
SEED_DEMO_FORCE=true npm run seed:demo-courses
```

Требуется применённая схема БД (`npm run migration:run`).
